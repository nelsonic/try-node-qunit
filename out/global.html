getChange(totalPayable, cashPaid) → {array}

getChange accepts two parameters (totalPayable and cashPaid) and calculates the change in "coins" that needs to be returned.
Parameters:
Name Type Description
totalPayable number the integer amount (in pennies) to be paid
cashPaid number the integer amount (in pennies) the person paid
Source:
  • change.js, line 14
Returns:
list of coins we need to dispense to the person as change
Type
array
Example
getChange(215, 300); // returns [50, 20, 10, 5]
